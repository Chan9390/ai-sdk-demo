# Project Instructions

## Overview

Demo project showcasing **Vercel AI SDK** (frontend) with **Langchain** and **AI SDK** backends.

## Code Style

- Use TypeScript
- No `any` types
- Keep code clear and concise

## Code Suggestion

- Never install npm libraries, give accurate commands so it can be manually executed by user

## Library Versions

### Langchain / Langgraph
- Version: **1.0.0+**
- ⚠️ Breaking changes exist from older versions
- Always verify you're reading v1 documentation

### AI SDK
- Version: **5.x**
- ⚠️ Do NOT use v6 (currently in beta)
